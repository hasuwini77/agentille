# Per-repo CLAUDE.md template

Render with values from the per-project answers + the global profile. Sections in **bold** are conditional on the field being non-empty.

```markdown
# <project.name>

<project.description?>

## Context

**Stack:** <project.techStack.join(", ")>
**Goal:** <project.goals?>
**How I use Claude here:** <project.claudeUse?>

## Constraints

<project.constraints?>

## Voice

Apply <profile.name>'s global communication preferences: <profile.deliveryStyle> delivery, <profile.tone> tone, <profile.honestyLevel> honesty.

**Never:**
- <profile.neverDo[i]>
- ...
- <profile.customNeverDo?>

---

*Generated by [agentille](https://github.com/hasuwini77/agentille) — edit freely.*
```

## Rendering rules

- Skip any section whose values are empty/blank
- "Stack" line is omitted if `techStack` is empty
- "Never:" list is omitted if both `neverDo` and `customNeverDo` are empty
- Trim trailing whitespace; end file with a single newline
